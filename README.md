# Joonas Virmajoki homepage

**Live website:** https://joonasvirmajoki.me/

---

#### Tools used

- Material-UI
- React.js
- React-Particles
- React-Typed
- React-CountUp

#### Features

- Timeline to show resume
- Place to show projects
- Contact info in Footer
- Togglable side navigation
- Mobile responsive
